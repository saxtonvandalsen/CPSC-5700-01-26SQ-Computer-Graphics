# CPSC-5700-01-26SQ-Computer-Graphics

## Catalog Course Description

The course presents modern graphics programming techniques in the form of in-class demonstrations and weekly assignments that develop skill in the representation, manipulation, and display of 2D and 3D objects.

## Course Learning Outcomes

1. Review the evolution of computer graphics
2. Examine GPU shader architecture
3. Compare ray-tracing and rasterization
4. Examine representation, transformation, and display
5. Demonstrate course topics with custom software
6. Develop a modest proficiency in OpenGL/GLSL
7. Practice memory management                                                                                    

## Topics

- vertex buffering and the graphics pipeline
- animation and interaction
- transformations and matrices
- projection and perspective
- meshes and shading
- parametric curves and surfaces
- sampling and anti-aliasing
- texture and other maps
- tessellation and level of detail
- alternate representation and display
